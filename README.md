# RSVP Form

A single-page RSVP form with a cream, botanical, vintage style. Submissions are sent via **Formspree** to the endpoint set in `index.html` (`data-api`). In your Formspree dashboard, set the email where you want to receive responses.

No backend or server is required: open `index.html` in a browser or host the folder on any static host (GitHub Pages, Netlify, etc.).
